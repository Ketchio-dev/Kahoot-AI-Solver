# Kahoot AI Solver - Smart Assistant

이 프로젝트는 크롬 확장 프로그램 **"Kahoot AI Solver"**의 소스 코드입니다.
AI 모델(Gemini, GPT)을 활용하여 카훗 퀴즈를 실시간으로 분석하고 정답을 알려주는 스마트 어시스턴트입니다.

## 🚀 주요 기능

- **다양한 AI 모델 지원**:
  - ⚡ **Gemini Flash**: 빠르고 효율적인 모델
  - 🧠 **Gemini Pro**: 더 정확하고 심층적인 분석
  - 🤖 **GPT-5.2**: OpenAI 최신 모델 (설정 필요)
- **스텔스 모드**:
  - 화면에 정답을 크게 표시하지 않고, 우측 하단에 작게 표시하여 눈에 띄지 않게 정답 확인 가능
- **단축키 지원**:
  - `Alt+Z`: Gemini Flash로 풀기
  - `Alt+X`: Gemini Pro로 풀기
  - `Alt+C`: GPT-5.2로 풀기

## 🛠️ 개발 설정

이 저장소를 클론하여 로컬에서 수정하거나 테스트할 수 있습니다.

1. 저장소 클론:
   ```bash
   git clone https://github.com/Ketchio-dev/Kahoot-AI-Solver.git
   ```
2. 크롬 확장 프로그램 관리 페이지(`chrome://extensions/`) 접속
3. 우측 상단 **"개발자 모드"** 켜기
4. **"압축해제된 확장 프로그램을 로드합니다"** 클릭 후 폴더 선택
